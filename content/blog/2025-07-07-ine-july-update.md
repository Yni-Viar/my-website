# Little Update for Demo 1.

This old demo, that was made back in February-March 2025, received an little update.

- Added dialogue icons (present at upcoming Demo 2)
- Switched default renderer to OpenGL ES (due to Godot bug)
- Made game lore up-to-date. (some content was using old lore)
- Small changes from Demo 2.

# Demo 2 development update.

Save system is now working. Actually, there is no checkpoints, where you save, instead it saves, when you switch *a location*.

Locations are separated with loading screens, and in that moment you are saving the game.
